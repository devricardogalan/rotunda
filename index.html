This is my url parser.

The point of this parser, is first to insert a format like so:

'/:version/api/:collection/:id'

And then a url instance like this one:

'/6/api/listings/3?sort=desc&limit=10'

After all of this is done it would return an object like this:

{ version:6, collection:'listings', id:3, sort:'desc', limit:10 }

Now please check your console for your result!
